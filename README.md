## Project Details
A simple sport's blog featuring top football leagues in Africa, Europe and The Americas. Relies on JSON server for the handling of storing data. Emulates CRUD functionalities.

## Technologies Used
- React
- CSS
- MDB 5
- Cloudinary
- JSON server
